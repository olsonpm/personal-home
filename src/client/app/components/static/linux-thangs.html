Bash

Here is my small Bash library. My first goal when learning bash was to create a usable 'import' command for use in other libraries. Once finished, I created a few import'able libraries including a logger, a test library, a library to manage configuration files, and a simple Bash file_exists wrapper.

VSFTPd

Here are my vsftpd scripts. When I first installed Ubuntu server I wanted an ftps server (I guess sftp is superior, but I already started working toward ftps before realizing this). Utilizing the above Bash libraries, I created the scripts which basically formulate an ftp server application. My requirements were:

  • Security comes first (hence vsftpd and my small contributions to pam-pgsql*)
  • Easily support a varying number of users (Scripting user CRUD was a must)
  • Users can belong to groups, where a group has a shared r/w directory as well as a read-only directory

Although I became comfortable working with Bash after this project, It definitely taught me why people choose python/ruby/etc when scripting small applications such as this one. Still, I was very happy with the outcome.

*Ubuntu's pam-pgsql package didn't support b-crypt and its other options were terrible in comparison, which is why I found an external repository with a b-crypt port.
